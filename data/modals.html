Mehgine

The Mehgine is a game engine made in C++ using OpenGL. With the help of several external libraries the engine has a set of subsistems, such as Unity like game objects and components, quadtrees, resource manangers and many more. On top of that the engine features a particle subsystem specialization.

The Heist

Prototype of a 3D level for a game concept made in Unreal Engine.
The main story of the level and the game is that the main character has to enter the minds of people to steal memories of information. The mind of the host is represented in a surrealistic environment where the memory to steal is a physical object. To reinforce the surrealistic mood, the closer to the object the smaller the player is.

Pepa and the Cookie bandit

Video Game Design and Development Degree Bachelor's Tesis This project is the study of the detection of disabilities through video games. See if it’s possible to detect learning disabilities.The game detects alert signals in the language for children from age 3 to 5.
The game consists in a set of challenges that analyze and test the player’s ability to perceive spoken language, how much knowledge they already got and if they express correctly in a conversation. At the end of the game session it will generate a document that their tutors will be able to view, in this document it will appear all the results from the challenges and how the player had performed.
